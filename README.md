# Student Management System

A console-based Student Management System developed in Java. This project demonstrates the implementation of Object-Oriented Programming (OOP), CRUD operations, file handling, sorting, and input validation.

## Features

- Add Student
- View Student Records
- Update Student Details
- Delete Student Records
- Search Student
- File Handling using text files
- Input Validation
- Sorting of Student Records

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File Handling
- Collections Framework

## Project Structure

```
Student-Management-System/
│── Main.java
│── Student.java
│── StudentManager.java
│── students.txt
│── README.md
```

## How to Run

1. Clone the repository.
2. Open the project in your Java IDE (VS Code, IntelliJ IDEA, or Eclipse).
3. Compile the Java files.
4. Run `Main.java`.
